The `GoogleMap` component is the core component of React Google Maps API. It creates and manages a Google Maps instance and provides context for all child components like markers, overlays, and other map features.

## Basic Usage

```jsx
import { GoogleMap, LoadScript } from '@react-google-maps/api'

const containerStyle = {
  width: '100%',
  height: '400px'
}

const center = {
  lat: 37.7749,
  lng: -122.4194
}

function MyMap() {
  return (
    <LoadScript googleMapsApiKey="YOUR_API_KEY">
      <GoogleMap
        mapContainerStyle={containerStyle}
        center={center}
        zoom={12}
      >
        {/* Add markers, overlays, etc. here */}
      </GoogleMap>
    </LoadScript>
  )
}
```

## Props

### Container Props

<ParamField path="id" type="string" optional>
  HTML element ID for the map container div
</ParamField>

<ParamField path="mapContainerStyle" type="CSSProperties" optional>
  Inline styles for the map container. Must include width and height.
  
  ```jsx
  mapContainerStyle={{ width: '100%', height: '400px' }}
  ```
</ParamField>

<ParamField path="mapContainerClassName" type="string" optional>
  CSS class name for the map container div
</ParamField>

### Map Configuration

<ParamField path="options" type="google.maps.MapOptions" optional>
  Complete Google Maps MapOptions object. This is the most flexible way to configure the map.
  
  ```jsx
  options={{
    disableDefaultUI: true,
    zoomControl: true,
    mapTypeControl: false,
    styles: [...] // Custom map styles
  }}
  ```
</ParamField>

<ParamField path="center" type="google.maps.LatLng | google.maps.LatLngLiteral" optional>
  Map center position. Can be a LatLng object or literal with lat/lng properties.
  
  ```jsx
  center={{ lat: 37.7749, lng: -122.4194 }}
  ```
</ParamField>

<ParamField path="zoom" type="number" optional>
  Map zoom level (0-22). Higher numbers show more detail.
  
  ```jsx
  zoom={12}
  ```
</ParamField>

<ParamField path="heading" type="number" optional>
  Map heading in degrees for 45° imagery (0-360)
</ParamField>

<ParamField path="tilt" type="number" optional>
  Map tilt in degrees for 45° imagery (0 or 45)
</ParamField>

<ParamField path="mapTypeId" type="string" optional>
  Map type identifier. Common values:
  - `'roadmap'` - Default road map view
  - `'satellite'` - Satellite imagery
  - `'hybrid'` - Satellite imagery with roads
  - `'terrain'` - Terrain elevation data
</ParamField>

<ParamField path="clickableIcons" type="boolean" optional default="true">
  Whether default map POI icons are clickable
</ParamField>

<ParamField path="streetView" type="google.maps.StreetViewPanorama" optional>
  Street View panorama instance to bind to the map
</ParamField>

<ParamField path="extraMapTypes" type="google.maps.MapType[]" optional>
  Additional map types to register with the map
</ParamField>

### Mouse Events

<ParamField path="onClick" type="(e: google.maps.MapMouseEvent) => void" optional>
  Fired when the user clicks on the map
  
  ```jsx
  onClick={(e) => {
    console.log('Clicked at:', e.latLng?.toJSON())
  }}
  ```
</ParamField>

<ParamField path="onDblClick" type="(e: google.maps.MapMouseEvent) => void" optional>
  Fired when the user double-clicks on the map
</ParamField>

<ParamField path="onRightClick" type="(e: google.maps.MapMouseEvent) => void" optional>
  Fired when the user right-clicks on the map
</ParamField>

<ParamField path="onMouseMove" type="(e: google.maps.MapMouseEvent) => void" optional>
  Fired when the mouse moves over the map
</ParamField>

<ParamField path="onMouseOver" type="(e: google.maps.MapMouseEvent) => void" optional>
  Fired when the mouse enters the map
</ParamField>

<ParamField path="onMouseOut" type="(e: google.maps.MapMouseEvent) => void" optional>
  Fired when the mouse leaves the map
</ParamField>

<ParamField path="onMouseDown" type="(e: google.maps.MapMouseEvent) => void" optional>
  Fired when the user presses the mouse button on the map
</ParamField>

<ParamField path="onMouseUp" type="(e: google.maps.MapMouseEvent) => void" optional>
  Fired when the user releases the mouse button on the map
</ParamField>

### Drag Events

<ParamField path="onDrag" type="() => void" optional>
  Fired repeatedly during map drag
</ParamField>

<ParamField path="onDragStart" type="() => void" optional>
  Fired when drag starts

<ParamField path="onDragEnd" type="() => void" optional>
  Fired when drag ends

### Map Property Change Events

<ParamField path="onBoundsChanged" type="() => void" optional>

<ParamField path="onCenterChanged" type="() => void" optional>
  Fired when the map center changes
</ParamField>

<ParamField path="onZoomChanged" type="() => void" optional>
  Fired when the zoom level changes
</ParamField>

<ParamField path="onMapTypeIdChanged" type="() => void" optional>
  Fired when the map type changes
</ParamField>

<ParamField path="onHeadingChanged" type="() => void" optional>
  Fired when the map heading changes
</ParamField>

<ParamField path="onTiltChanged" type="() => void" optional>
  Fired when the map tilt changes
</ParamField>

<ParamField path="onProjectionChanged" type="() => void" optional>
  Fired when the map projection changes
</ParamField>

### Other Events

<ParamField path="onIdle" type="() => void" optional>
  Fired when the map becomes idle after panning or zooming
</ParamField>

<ParamField path="onTilesLoaded" type="() => void" optional>
  Fired when the visible tiles have finished loading
</ParamField>

<ParamField path="onResize" type="() => void" optional>
  Fired when the map div changes size
</ParamField>

### Lifecycle Callbacks

<ParamField path="onLoad" type="(map: google.maps.Map) => void | Promise<void>" optional>
  Callback invoked when the map is loaded. Receives the map instance.
  
  ```jsx
  onLoad={(map) => {
    console.log('Map loaded:', map)
    // Store map instance in ref or state
  }}
  ```
</ParamField>

<ParamField path="onUnmount" type="(map: google.maps.Map) => void | Promise<void>" optional>
  Callback invoked before the map is unmounted. Receives the map instance.
  
  ```jsx
  onUnmount={(map) => {
    console.log('Map unmounting:', map)
    // Cleanup logic here
  }}
  ```
</ParamField>

<ParamField path="children" type="ReactNode" optional>
  Child components like Marker, Polyline, InfoWindow, etc.
</ParamField>

## Examples

### Controlled Map with State

```jsx
import { useState } from 'react'
import { GoogleMap, LoadScript } from '@react-google-maps/api'

function ControlledMap() {
  const [center, setCenter] = useState({ lat: 37.7749, lng: -122.4194 })
  const [zoom, setZoom] = useState(12)

  return (
    <LoadScript googleMapsApiKey="YOUR_API_KEY">
      <GoogleMap
        mapContainerStyle={{ width: '100%', height: '400px' }}
        center={center}
        zoom={zoom}
        onCenterChanged={() => {
          // Note: Get center from map instance stored in ref
        }}
        onZoomChanged={() => {
          // Note: Get zoom from map instance stored in ref
        }}
      />
    </LoadScript>
  )
}
```

### Accessing Map Instance

```jsx
import { useRef } from 'react'
import { GoogleMap, LoadScript } from '@react-google-maps/api'

function MapWithInstance() {
  const mapRef = useRef(null)

  const onLoad = (map) => {
    mapRef.current = map
  }

  const handlePanTo = () => {
    if (mapRef.current) {
      mapRef.current.panTo({ lat: 40.7128, lng: -74.0060 })
    }
  }

  return (
    <>
      <button onClick={handlePanTo}>Pan to New York</button>
      <LoadScript googleMapsApiKey="YOUR_API_KEY">
        <GoogleMap
          mapContainerStyle={{ width: '100%', height: '400px' }}
          center={{ lat: 37.7749, lng: -122.4194 }}
          zoom={12}
          onLoad={onLoad}
        />
      </LoadScript>
    </>
  )
}
```

const mapStyles = [

### Handling Click Events

```jsx
import { useState } from 'react'
import { GoogleMap, Marker, LoadScript } from '@react-google-maps/api'

function InteractiveMap() {
  const [markers, setMarkers] = useState([])

  const handleMapClick = (e) => {
    const newMarker = {
      lat: e.latLng.lat(),
      lng: e.latLng.lng()
    }
    setMarkers([...markers, newMarker])
  }

  return (
    <LoadScript googleMapsApiKey="YOUR_API_KEY">
      <GoogleMap
        mapContainerStyle={{ width: '100%', height: '400px' }}
        center={{ lat: 37.7749, lng: -122.4194 }}
        zoom={12}
        onClick={handleMapClick}
      >
        {markers.map((marker, index) => (
          <Marker key={index} position={marker} />
        ))}
      </GoogleMap>
    </LoadScript>
  )
}
```

## Best Practices

### 1. Memoize Container Style

Define the container style outside the component or use `useMemo` to prevent unnecessary re-renders:

```jsx
const containerStyle = {
  width: '100%',
  height: '400px'
}

// Or inside component:
const containerStyle = useMemo(() => ({
  width: '100%',
  height: '400px'
}), [])
```

### 2. Store Map Instance in Ref

Use a ref to store the map instance for programmatic control:

```jsx
const mapRef = useRef(null)

const onLoad = useCallback((map) => {
  mapRef.current = map
}, [])
```

### 3. Use Options Prop for Complex Configuration

For advanced configuration, use the `options` prop instead of individual props:

```jsx
const options = useMemo(() => ({
  disableDefaultUI: true,
  zoomControl: true,
  styles: mapStyles,
  gestureHandling: 'cooperative'
}), [])

<GoogleMap options={options} />
```

### 4. Debounce Expensive Operations

Debounce callbacks for events that fire frequently like `onBoundsChanged` or `onDrag`:

```jsx
import { useMemo } from 'react'
import debounce from 'lodash/debounce'

const handleBoundsChanged = useMemo(
  () => debounce(() => {
    // Expensive operation
  }, 300),
  []
)
```

### 5. Clean Up Event Listeners

The component automatically cleans up all event listeners, but if you add custom listeners via the map instance, clean them up in `onUnmount`:

```jsx
const onLoad = (map) => {
  const listener = map.addListener('click', handleClick)
  // Store listener to remove later
}

const onUnmount = (map) => {
  // Remove custom listeners
}
```

## Common Patterns

### Responsive Map Container

```jsx
const containerStyle = {
  width: '100%',
  height: '100vh'
}

// Or with CSS:
// .map-container { width: 100%; height: 100vh; }

<GoogleMap
  mapContainerClassName="map-container"
  center={center}
  zoom={12}
/>
```

### Fitting Bounds to Markers

```jsx
const onLoad = (map) => {
  const bounds = new window.google.maps.LatLngBounds()
  markers.forEach(marker => {
    bounds.extend(marker.position)
  })
  map.fitBounds(bounds)
}
```
